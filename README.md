# 🛠️ Multi Tools

Kumpulan tools online gratis untuk developer, desainer, dan pengguna umum. Tanpa login, langsung pakai.

## Tools yang Tersedia

- **Text Tools** — Case Converter, Text Cleaner, Word Counter, Base64 Encode/Decode
- **Image Tools** — Image Compressor, Resizer, Format Converter, Cropper
- **Developer Tools** — JSON Formatter, HTML Beautifier, UUID & Password Generator
- **Lainnya** — QR Generator, Color Picker, Unit Converter, Time Zone Converter

## Cara Pakai

Clone repositori lalu buka `index.html` di browser — tidak perlu instalasi apapun.

```bash
git clone https://github.com/username/multi-tools.git
cd multi-tools
open index.html
```

## Menambah Tool Baru

1. Buat folder `tools/nama-tool/` dan isi dengan `index.html`
2. Tambahkan card di `index.html` utama
3. Tambahkan link di dropdown navbar yang sesuai

## Teknologi

HTML · CSS · JavaScript — tanpa framework, tanpa dependensi.
